## FAQs

**Q: Do I need to create a new saved group every time?**
A: No! Create it once, use it forever. Just click "Start Session" each time.

**Q: Why do I need to enter my name when starting a session?**
A: The session is live for tonight. You're creating the group and inviting others to join with the code.

**Q: Can my friends see my saved groups?**
A: No, saved groups are private. Friends join by entering the session code you share.

**Q: What if I have different friend groups?**
A: Create multiple saved groups! "Friday Crew", "Work Lunch", "Family Dinners", etc.

**Q: Do saved group members auto-join sessions?**
A: No - you still need to share the session code. Members use their own devices to join and swipe.

**Q: Can I edit a saved group?**
A: Yes! Click "Edit" on any saved group to update preferences or members.
